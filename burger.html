Burger Builder

Front End - React Project

Description

A Burger ordering website built with ReactJS.
User has the ability to signup, signin, design, edit and place an order to his created burger.
User has to signin to be able to view his/her orders and be able to place an order.
The user can add his/her address information for delivery and can choose from different delivery options.

Technologies Used

  • ReactJS
  • Firebase
  • Redux
  • Axios
  • Bootstrap
  • HTML
  • CSS
  • Javascript
