DevBook

MERN Stack Project

Description

A social network for developers - A full stack web application, built with MERN stack technologies.
User can have a detailed profile includes all of his/her personal information, experiences and educations.
User can add posts and and share it with the other users. Also can like, unlike and comment to other users' posts.
Can also add his/her experiences and educations, user can add his/her github user name to get the latest 5 created github repositories.
Also can connect his/her social links like facebook, twitter, linkedin, instagram, and youtube.

Technologies Used

  • NodeJS
  • ReactJS
  • Redux
  • Axios
  • MongoDB
  • Express
  • Bootstrap
  • HTML
  • CSS

Note: A user need to sign up with a connected to gravatar email to be able to have a profile picture.
